trace local context cloud

An OS-level memory layer that captures computer activity and indexes it with Supermemory for AI-powered recall.

Architecture
Daemon
Persistent PowerShell tracker captures foreground windows, browser URLs, process snapshots, and idle state on a 3-second loop. File watcher monitors create/edit events. Editor detector identifies active projects.
Storage
All events written to ~/.trace/events.jsonl and fire-and-forget posted to Supermemory Local (localhost:6767) for vector indexing and hybrid search. Falls back to local-only when Supermemory is unreachable.
API
Express server on port 6768 with REST endpoints for context, search, and daily summaries. MCP-compatible tools for AI agent integration. LLM-powered Q&A via any OpenAI-compatible endpoint.
Overlay
Frameless Electron window (Alt+X) with chat-style search bar and activity feed. Web-based HUD dashboard on port 6769 for live session tracking.
